WINDOW语句

WINDOW语句 - 创建一个新的窗口。

语法​

WINDOW name [caption] [options];

语句末尾出现的options可以按任意顺序一个接一个地指定：

HIDETITLE 
HIDESCROLLBARS 
orientationType
POSITION(x, y, width, height)
fixedPositionType
HALIGN(alignType)
VALING(alignType) 
TEXTHALIGN(alignType)
TEXTVALIGN(alignType)
CLASS cssClassExpr

描述​

WINDOW语句声明一个新窗口并将其添加到当前模块。

参数​

  • name

    窗口名称。简单ID。该名称在当前命名空间内必须是唯一的。

  • caption

    窗口标题。字符串字面量。如果未指定标题，则将使用窗口名称作为标题。

选项​

  • HIDETITLE

    指定在用户界面中不显示标题的关键字。

  • HIDESCROLLBARS

    指定为此窗口不显示滚动条的关键字。

  • orientationType

    指定正在创建的工具栏或面板的垂直或水平方向。通过以下关键字之一指定：

    • VERTICAL - 垂直方向（默认值）。
    • HORIZONTAL - 水平方向。

  • POSITION (x, y, width, height)

    指定窗口的大小和位置。

    • x

      窗口左坐标。范围从0到100的整数字面量。

    • y

      窗口上坐标。范围从0到100的整数字面量。

    • width

      窗口宽度。范围从0到100的整数字面量。

    • height

      窗口高度。范围从0到100的整数字面量。

  • fixedPositionType

    指定窗口在桌面上的固定位置，不允许用户更改其位置和大小。在这里，窗口大小是根据组件的首选尺寸自动确定的。窗口将分别位于桌面的左侧、右侧、顶部和底部。此选项不能与POSITION选项同时使用。通过以下关键字之一指定：

    • LEFT
    • RIGHT
    • TOP
    • BOTTOM

  • HALIGN(alignType)

    指定垂直工具栏中按钮的水平对齐方式。

    • alignType

      对齐类型。通过以下关键字之一指定：

      • START - 左对齐（默认值）。
      • CENTER - 居中对齐。
      • END - 右对齐。

  • VALIGN(alignType)

    指定水平工具栏中按钮的垂直对齐方式。

    • alignType

      对齐类型。通过以下关键字之一指定：

      • START - 顶部对齐（默认值）。
      • CENTER - 居中对齐。
      • END - 底部对齐。

  • TEXTHALIGN(alignType)

    指定按钮上文本的水平对齐方式。

    • alignType

      对齐类型。通过以下关键字之一指定：

      • START - 左对齐（默认值）。
      • CENTER - 居中对齐。
      • END - 右对齐。

  • TEXTVALIGN(alignType)

    指定按钮上文本的垂直对齐方式。

    • alignType

      对齐类型。通过以下关键字之一指定：

      • START - 顶部对齐。
      • CENTER - 居中对齐（默认值）。
      • END - 底部对齐。

  • CLASS cssClassExpr

    指定为HTML中窗口组件创建的DOM元素的CSS类名。这可用于应用自定义样式。

    • cssClassExpr

      表达式，其值确定类名。

示例​

// 在System模块中创建系统窗口
WINDOW logo HORIZONTAL POSITION(0, 0, 10, 6) VALIGN(CENTER) HALIGN(START) HIDETITLE HIDESCROLLBARS CLASS logoWindowClass();
WINDOW root HORIZONTAL POSITION(10, 0, 70, 6) VALIGN(CENTER) HALIGN(CENTER) HIDETITLE HIDESCROLLBARS CLASS rootWindowClass();
WINDOW system HORIZONTAL POSITION(80, 0, 20, 6) VALIGN(CENTER) HALIGN(END) HIDETITLE HIDESCROLLBARS CLASS systemWindowClass();

WINDOW toolbar VERTICAL POSITION(0, 6, 20, 94) HIDETITLE CLASS toolbarWindowClass();

// 桌面底部的水平工具栏，其中所有按钮将居中，文本将向上对齐
// 在这个工具栏中，例如，可以放置用于快速打开的表单
WINDOW hotforms HORIZONTAL BOTTOM VALIGN(CENTER) TEXTVALIGN(START);
