ASK 操作符

ASK 操作符创建一个动作，以对话框形式向用户显示消息。

语法​

ASK expression 
[[alias =] YESNO]
[DO actionOperator [ELSE elseOperator]]

描述​

ASK 操作符创建一个动作，用于向用户请求确认/向用户提问。

参数​

  • expression

    一个表达式，其值为消息字符串。如果值为 NULL，则不会提出问题，但动作仍将执行。

  • YESNO

    如果指定，则向用户提出一个问题（是/否）

  • alias

    本地参数的名称，用户对问题的回答将写入其中（是 = TRUE，否 = NULL）。简单 ID。

  • actionOperator

    一个上下文相关的动作操作符，如果输入成功完成，则执行该操作符。可以使用上层参数和用户响应参数（如果问题是是/否）作为参数

  • elseActionOperator

    一个上下文相关的动作操作符，如果输入被取消，则执行该操作符。只能使用上层参数作为参数。

示例​

testAsk ()  {
    ASK 'Are you sure you want to continue?' DO {
        MESSAGE 'You continued';
    }

    ASK 'Use old values?' useOld = YESNO DO {
        IF useOld THEN
            MESSAGE 'Using old values';
        ELSE
            MESSAGE 'Using new values';
    }
}
