数据属性 (DATA)

数据属性是一种属性，其值存储在数据库中，并可通过执行相应的操作来更改。数据属性的每个参数和值必须属于某个指定的类。如果参数不属于指定的类或是 NULL，则属性值将返回 NULL。

局部数据属性​

数据属性可以是局部的。此类属性仅在会话内保留其值，即它们不会保存到数据库中，这意味着在应用更改时，这些值默认会重置为 NULL。

语言​

要声明主要属性，请使用DATA 操作符。

示例​

CLASS Item;
quantity = DATA LOCAL INTEGER (Item);

CLASS Country;
isDayOff = DATA BOOLEAN (Country, DATE);
